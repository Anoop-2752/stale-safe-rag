# IT Security Policy

All employees must use company-approved password managers and enable two-factor authentication on all work accounts.

Company laptops must have full-disk encryption enabled at all times. Personal devices are not permitted to access internal systems without prior IT approval.

Any suspected security incident, including phishing emails, must be reported to the IT security team within 24 hours of discovery.

Passwords must be rotated every 90 days and must not be reused across the last 5 password cycles.
